1) 페이지 구조
/krw-to-coin
원화 → 코인

입력한 KRW 금액이

스테이블 코인 + 일반 코인 각각 몇 개인지

한 번에 목록으로 보여줌

/coin-to-krw
코인 → 원화

입력한 스테이블 코인(USDT, USDC 등) 개수를

KRW로 환산한 결과만 보여줌

두 페이지만 만들어 놓고, 네비게이션 바 로 간단히 왔다 갔다 할 수 있게.

2) krw-to-coin 화면 와이어프레임
복사
편집
┌──────────────────────────────────────────┐
│ ← 환전 계산기 │ KRW→Coin │ Coin→KRW →   │  ← 탭바
├──────────────────────────────────────────┤
│ 💰 원화 입력                           │
│   ┌──────────────┐  ┌─────────┐         │
│   │  금액 입력  │  │  변환 버튼 │         │
│   │  (예: 10,000)│  └─────────┘         │
│   └──────────────┘                       │
├──────────────────────────────────────────┤
│ 🔄 변환 결과 (자동 갱신 혹은 버튼 클릭) │
│  ┌──────────────────┬─────────────┐      │
│  │ 코인 심볼         │ 수량        │      │
│  ├──────────────────┼─────────────┤      │
│  │ USDT             │ 7.19 개     │      │
│  │ USDC             │ 7.18 개     │      │
│  │ BTC              │ 0.000126 BTC│      │
│  │ ETH              │ 0.00175 ETH │      │
│  └──────────────────┴─────────────┘      │
│ *1 USD = 1,392 KRW (네이버),            │
│  USDT=1.0002 USD, BTC=57,000 USD (CG)    │
└──────────────────────────────────────────┘
입력란: 숫자만

변환 결과: 미리 정해둔 코인 리스트(스테이블 + 일반)를 순서대로 보여줌

출처 표시: 작은 폰트로 API 이름 + 마지막 업데이트 시간

3) coin-to-krw 화면 와이어프레임
복사
편집
┌───────────────────────────────────┐
│ ← 환전 계산기 │ KRW→Coin │ Coin→KRW → │  
├───────────────────────────────────┤
│ 🪙 코인 선택 & 개수 입력        │
│  ┌───────────┬───────────┐      │
│  │ 드롭다운  │  수량 입력 │      │
│  │ (USDT)    │ (예: 10)   │      │
│  └───────────┴───────────┘      │
│  └── [환산 버튼] ──┘            │
├───────────────────────────────────┤
│ 🔄 변환 결과                      │
│  ┌──────────────────────────┐     │
│  │ 10 USDT = 13,920 KRW     │     │
│  └──────────────────────────┘     │
│ *1 USD = 1,392 KRW (네이버)       │
└───────────────────────────────────┘
드롭다운: 스테이블 코인 목록만

결과: “X 코인 = Y 원” 단일 문장
